# Project Instructions

This is an Electron desktop pet app.

- Keep generated code commented where the behavior is not obvious.
- Do not commit `node_modules/`, build output, or temporary OS files.
- New pet actions should be added by creating a folder under `cat_anime/flames/`.
- Each action folder should contain an ordered image sequence, for example `01_no_bg.png` through `24_no_bg.png`.
- The app should continue to discover action folders automatically without requiring code changes for each new action.
- Use `npm start` to run the Electron app locally.
